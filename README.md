Yang Yang's personal website, generated by [jemdoc](http://jemdoc.jaboc.net/).

## Build

```
./build.py
```

This runs jemdoc and then patches a few things jemdoc cannot emit directly (HTML5 doctype, `lang="en"`, `<main>` landmark, `<h1>`, deprecated-attribute strip, list coalescing, inline tag spans). Requires only `python3`.

The bare `python3 jemdoc.py -c site.conf index.jemdoc` also produces a deployable site, but skips the polish; the output validates as XHTML 1.1 instead of HTML5 and scores lower on accessibility/SEO audits.

## Edit

- `index.jemdoc` — page content
- `site.conf` — site-wide config (head meta, fonts, JSON-LD, favicons)
- `jemdoc.css` — styling, mobile breakpoint, dark mode, tag chips
- `build.py` — post-processing rules (add a new transform here if jemdoc emits something you want to clean up)

After editing source, run `./build.py` and commit the regenerated `index.html` alongside your source changes.

## Preview

```
python3 -m http.server 8000
```

Open <http://localhost:8000/index.html>.

## Deploy

Commit and push to `master`. GitHub Pages serves <https://st2yang.github.io/> from the repo root within ~60 seconds.
